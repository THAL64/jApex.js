# How to Use

1. jquery.jApex.js を任意のディレクトリに保存
1. jQuery定義後に読み込み
```html:sample
<head>
    <script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>
    <script src="path/to/files/jquery.jApex.js"></script><!-- ←保存した jquery.jApex.js へのパス -->
</head>
<body>
    ……
</body>
```
